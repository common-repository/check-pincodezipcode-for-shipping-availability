== Description ==

**Woocommerce plugin to check shipping availability using pincode ** using this plugin you can check the shipping is available or not in a particular postal code or zip code. 

**Pincode Checker ** function will provide you the option to add how many days in delivery get as well Cash on delivery Available or not into Product Woocommerce. You can also change the label text using the admin panel.

**Cash on Delivery Amount ** You can add cash on delivery (COD) amount on each pincode / Zipcode separately. COD amount will display on cart under cart totals and checkout page under order details.

== Installation ==

1. Upload the directory `/check-pincode-zipcode-for-shipping-availability/` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Click on "Post/Zip Code" menu and import post/zip code and manage the setting.
